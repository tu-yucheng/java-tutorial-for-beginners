# Introduction

本站点主要记录Java生态系统中常用框架或第三方库的笔记，
尤其是Java核心、Web开发、Spring框架全家桶系统、持久层框架、测试库、Apache项目、数据结构与算法、设计模式。  
所有的文章都有对应的源代码和测试代码。你可以克隆该项目并直接在你的机器上运行。

# Getting Started

该项目使用`Java 17` + `Maven` + `Intellij IDEA 2022.2`构建。  
请确保你的本地安装了`JDK 17`并且配置好环境变量。
